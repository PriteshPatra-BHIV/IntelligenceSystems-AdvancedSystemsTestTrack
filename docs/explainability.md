# Explainability Layer

## Purpose
Allow the system to justify decisions without human presence.

## Every Explanation Answers
1. What action was taken
2. Why it was taken
3. What evidence supported it
4. What is still unknown

## Non-Claims
The system never claims:
- Certainty
- Optimality
- Complete knowledge

## Outcome
Decisions are auditable, replayable,
and bounded by known limits.
